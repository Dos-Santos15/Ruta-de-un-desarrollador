¿Como inicio?

Y si te lo preguntabas, yo te lo contesto. Si, si hay una manera buena de como iniciar una ruta para ser desarrollador web. Primeramente ennumeremos los pasos.

1. Nunca es malo entender como funciona un arte aun mas cuando vamos a entrar en el. Por eso recomiendo totalmente hacer un curso basico de logíca de programación. Asi a la hora de comenzar a programar se nos pueda hacer un poco mas facíl entender que hacemos con cada función.

2. Lo segundo pero tambien importante que debemos hacer es ver que area nos favorece mas. Actualmente existen 2 areas divididas, Frond-end y Back-end que conjuntamente forman al desarollador web Full-stack

3. Despúes de averiguar un poco de que van las areás y sus principios. lo mejor es buscar cursos basicos sobre sus lenguajes. En el areá Frond-end lo primero que debemos de aprender si o si es HTML, Css Y JavaScript. y en el mismo orden como esta escrito. para despues aprender los Framework de estos, como Bootstrap(Css), Sass(Css), React(JavasScript), Angular(JavaScript), Vue.js(JavaScript). Si lo tuyo es la parte interna de las paginas web, osea el areá Back-end los mas recomendable es que inicies con un lenguaje Back-end como JavaScript, Typescript, Phyton O Java, y especializarse totalmente en alguno de ellos, aprendiendo Frameworks y librerias. Mas adelante veremos como especializarse en JavaScript en el area Back-end.

4. Por último, antes de comenzar a practicar y estudiar en algunas de estas 2 areás, debemos instalar en nuestra PC un lector de codigó. En este ultimo año, el mejor y el mas usado ha sido Visual studio code. muy potente y con una gran variedad de extensiones para poder modificarlo a nuestro antojo

Frond-end

Si te interesa la creación digítal y visual de las pagínas web desde cero, este es tu lugar. aqui en Ruta de un desarollador hemos preparado tu mejor ruta para poder ser un gran Frond-end developer, Divinamos la ruta en pasos.

1.El paso número 1 no te lo puedes saltar por nada si en realidad deseas ser un desarollador frond-end, Siempre lo primero que debemos de aprender es HTML, Css y JavasScript. Html y css lo puedes aprender a un nivel intermedio/avanzado, asi se podra facilitar un poco mas las cosas y no te haga la vida imposible JavaScript. JavaScript con solo tener un nivel junior ya puedes pasar a sus Framework... Pero.

2. En un mundo laboral tan grande y donde se encuentra tanta competencia yo te recomiendo aprender un Nivel medio de JavaScript como segundo a seguir.

2.5 Es opcional pero no estaria mal y mas aun si eres bueno aprendiendo 2 cosas al mismo tiempo. despues de tener un nivel junior de JavaScript podrias ir aprendiendo conjuntamente un Framework como React.js mientras aprendes JavaScript aun nivel intermedio y asi podrias acortar tiempos.

3. Despues de las anteriores tienes que decidir en que Framework te vas a especializar, Actualmente los 3 pilares son React.js, Vue.js y Angular. Mi recomendación seria que te inclinaras por React.js, ya que actualmente es el que mas demanda tiene y es el lider actual entre las empresas
